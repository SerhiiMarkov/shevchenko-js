shevchenko

JavaScript library to inflect Ukrainian first, middle and last names

Demo

Fill the person's info in a nominative case.

Documentation

Installation
npm
npm install --save shevchenko
Node.js
const shevchenko = require("shevchenko");
Browser
<‍script type="text/javascript" src="/path/to/shevchenko.min.js"><‍/script>
Usage Example
const anthroponym = {
    gender: 'male', // or 'female'
    firstName: 'Тарас',
    middleName: 'Григорович',
    lastName: 'Шевченко'
};

const result = shevchenko.inVocative(anthroponym);

console.log(result); // { gender: 'male', firstName: 'Тарасе', middleName: 'Григоровичу', lastName: 'Шевченку' }
API
shevchenko.inNominative(anthroponym: Anthroponym): Anthroponym; // Inflect in nominative grammatical case
shevchenko.inGenitive(anthroponym: Anthroponym): Anthroponym; // Inflect in genitive grammatical case
shevchenko.inDative(anthroponym: Anthroponym): Anthroponym; // Inflect in dative grammatical case
shevchenko.inAccusative(anthroponym: Anthroponym): Anthroponym; // Inflect in accusative grammatical case
shevchenko.inAblative(anthroponym: Anthroponym): Anthroponym; // Inflect in ablative grammatical case
shevchenko.inLocative(anthroponym: Anthroponym): Anthroponym; // Inflect in locative grammatical case
shevchenko.inVocative(anthroponym: Anthroponym): Anthroponym; // Inflect in vocative grammatical case
