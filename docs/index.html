shevchenko

JavaScript бібліотека для відмінювання українських антропонімів

Демонстрація Приклад використання

Демонстрація Відмінювання українських прізвищ, імен та по батькові

Введіть стать, прізвище, ім'я та по батькові особи в називному відмінку однини.
Граматичний відмінок Прізвище Ім'я По батькові

Родовий (кого/чого?) Шевченка Тараса Григоровича

Принцип роботи

Бібліотека використовує більше вісімдесяти правил для відмінювання прізвищ, імен та по батькові. Правила фільтруються за родом, закінченням, частиною мови, сферою застосування та сортуються за пріорітетом. При відмінюванні жіночих прізвищ із закінченнями на -а, -я та чоловічих прізвищ із закінченнями на -ой, -ій, -ий, -их для визначення частини мови слова використовуються нейронні мережі прямого поширення (навчені на більш як сорока тисячах прізвищах). Тому прізвища іменникового та прикметникового походжень відмінюються як іменники та прикметники відповідно, наприклад, Диня (називний) -> Дині (родовий), Синя (називний) -> Синьої (родовий), Рий (називний) -> Рия (родовий), Бурий (називний) -> Бурого (родовий). При відмінюванні відбуваються зміни голосних та приголосних звуків згідно правил української мови, наприклад, Сивокінь (називний) -> Сивоконя (родовий), Гичка (називний) -> Гичці (давальний), але Чміль (називний) -> Чміля (родовий). Складені прізвища, утворені поєднанням двох самостійних прізвищ відмінюються за правилами відмінювання кожної складової частини, наприклад, Нечуй-Левицький (називний) - Нечуя-Левицького (родовий), але Драй-Хмара (називний) - Драй-Хмари (родовий), бо перша коротка частина складеного прізвища є односкладовою та сприймається не як самостійне прізвище, а як приставка до другого компонента. Складені імена, утворені поєднанням двох самостійних імен відмінюються за правилами відмінювання кожної складової частини, наприклад, Анна-Марія (називний) - Анни-Марії (родовий).

Документація

Встановлення
npm
npm install --save shevchenko
Node.js
const shevchenko = require("shevchenko");
Браузер 
<‍script type="text/javascript" src="/path/to/shevchenko.min.js"><‍/script>
Приклад використання 
const anthroponym = {
  gender: 'male', // or 'female'
  firstName: 'Тарас',
  middleName: 'Григорович',
  lastName: 'Шевченко'
};

const result = shevchenko.inVocative(anthroponym);

console.log(result); // { gender: "male", firstName: "Тарасе", middleName: "Григоровичу", lastName: "Шевченку" }
API
shevchenko.inNominative(anthroponym: Anthroponym): Anthroponym; // Провідміняти в називному відмінку
shevchenko.inGenitive(anthroponym: Anthroponym): Anthroponym; // Провідміняти в родовому відмінку
shevchenko.inDative(anthroponym: Anthroponym): Anthroponym; // Провідміняти в давальному відмінку
shevchenko.inAccusative(anthroponym: Anthroponym): Anthroponym; // Провідміняти в знахідному відмінку
shevchenko.inAblative(anthroponym: Anthroponym): Anthroponym; // Провідміняти в орудному відмінку
shevchenko.inLocative(anthroponym: Anthroponym): Anthroponym; // Провідміняти в місцевому відмінку
shevchenko.inVocative(anthroponym: Anthroponym): Anthroponym; // Провідміняти в кличному відмінку
